# Contributing

Thanks for your interest in improving Snake. This project keeps tooling minimal.

## Guidelines

- Keep dependencies at zero unless there is a strong reason.
- Prefer deterministic, testable logic for game rules.
- Keep UI minimal and consistent with existing style.

## Workflow

1. Create a branch.
2. Make changes.
3. Test manually in the browser.
4. Open a pull request with a short summary and a verification checklist.
