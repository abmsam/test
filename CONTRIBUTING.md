# Contributing

Thanks for your interest in improving Snake. This project keeps tooling minimal.

## Code Style

- Use plain JavaScript (ES2019+).
- Prefer small pure functions.
- Avoid global side effects outside initialization.
- Keep UI changes minimal and consistent.
- Use 2-space indentation in new files.

## Guidelines

- Keep dependencies at zero unless there is a strong reason.
- Prefer deterministic, testable logic for game rules.
- Keep UI minimal and consistent with existing style.

## Workflow

1. Create a branch.
2. Make changes.
3. Test manually in the browser.
4. Run CI checks if available.
5. Open a pull request with a short summary and a verification checklist.
